**Service :-

1.Service is basic building of angular.

2.It is used to separate UI logic from business logic. (i.e. It is the bridge between UI logic from business logic.)

3.Service is a simple class with methods and properties.

4.It is also used for inter-component communication.

5.Basically service is used with purpose of backend communication as well code reusability.

**Dependency Injection :-


It is a pattern of class coding in which class receives its dependencies externally rather than creating itself.



HTTP Client :-



1.HttpClient is a built in service used to communicate with backend.

2.It is also used for sending the http request and get the response.

3.We will get the reposonse of http request in terms of observable.

4.It can have error handling capability.

5.We can cancel the http request.

6.Htpp provides us below methods :-

    get ----> Used to get the data from the backend.
    post ----> Used to send as well get the data from backend.
    put ----> Used to update the data.
    delete---> Used to delete the data.
    **post ---> This method can perform functionality of all above methods.

**Observables :-


Observables are asynchronous or parellal flow of execution which can handle multiple events at a time.
