Angular Pipes


**Built in Pipes :-


1. Uppercase :-


Original text :- {{test1}}


*Uppercase text :- {{ test1 | uppercase }}

*Lowercase text :- {{ test1 | lowercase }}

*Titlecase text :- {{ test1 | titlecase }}

2. Date :-


*Original date :- {{ date }}

*Standard format :- {{ date | date }}

*Short date :- {{ date | date : "short" }}

*Medium date :- {{ date | date : "medium" }}

*Long date :- {{ date | date : "long" }}

*Full date :- {{ date | date : "full" }}

*Short date :- {{ date | date : "shortDate" }}

*Medium date :- {{ date | date : "mediumDate" }}

*Long date :- {{ date | date : "longDate" }}

*Full date :- {{ date | date : "fullDate" }}

*dd/mm/yyyy :- {{ date | date : "dd/MM/yyyy" }}

*Short time :- {{ date | date : "shortTime" }}

*Medium time :- {{ date | date : "mediumTime" }}

*Long Time :- {{ date | date : "longTime" }}

*Full Time :- {{ date | date : "fullTime" }}


3. Json :-


Json pipe :- {{person | json}}

Json pipe :- {{person.fname | json}}

Json pipe :- {{person.lname | json}}

Json pipe :- {{person.age | json}}


4. Slice pipe :-


Original text :- {{test1}}


*Uppercase text Slice :- {{ test1 | uppercase | slice : 0 :12 }}

*Lowercase text Slice :- {{ test1 | lowercase | slice : 3:6}}

*Titlecase text Slice :- {{ test1 | titlecase | slice : 0:10}}


5. Currency pipe :-


*India Rupee currency :- {{ salary | currency : "INR" }}

*United states doller currency :- {{ salary | currency : "USD" }}

*Viet Nam Dong currency :- {{ salary | currency : "VND" }}

*United Kingdom Pound currency :- {{ salary | currency : "GBP" }}

*Japan Yen currency :- {{ salary | currency : "JPY" }}

*Israel Shekel currency :- {{ salary | currency : "ILS" }}

*Brazil Real currency :- {{ salary | currency : "BRL" }}


6. Multiple pipes :-


*Multiple pipes :- {{ test2 | slice : 0:25 | uppercase }}
